## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/M-Taimoor-Mansha/Daily-Coding-Challenges.git
```

2. Navigate into the folder:

```
cd Daily-Coding-Challenges
```

3. Run any file:

```
node challenges/day-01.js
```
